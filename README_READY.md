

## Backend (API)
1. Open terminal and go to `backend` folder:
   ```
   cd backend
   ```
2. Install dependencies:
   ```
   npm install
   ```

3. Start backend in dev mode:
   ```
   npm run dev
   ```
   or for production:
   ```
   npm start
   ```

## Frontend (Vite)
1. Open new terminal and go to `frontend` folder:
   ```
   cd frontend
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start frontend:
   ```
   npm run dev
   ```
   The dev server typically runs at `http://localhost:5173`.
